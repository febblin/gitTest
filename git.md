﻿### How create a repository on GitHub?

1. Sign up with [GitHub](https://github.com/signup?ref_cta=Sign+up&ref_loc=header+logged+out&ref_page=%2F&source=header-home)

2. Select "New repository" ->  "New" -> add a name of repository and press "Create repository"

3. Do steps in Quick Setup page 

### If you want to copy a user's repository and create a new branch in that repository:

1. Select "Fork" in a user's repository 

2. Copy a link of a new repository  

3. Open a VS and git clone the links - create a copy of your repository or branch

4. git add [name a file] - add new or changed file 
<br> git commit - create a commit

5. git branch [new branch] - create a branch
<br> git checkout [new branch] - switch to the branch
<br> git add [name the file]
<br> git commit - create a commit

6. git push - upload local branch commits to the corresponding remote branch

7. git pull - update your current local branch
