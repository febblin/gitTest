﻿# Тута гит!

git --version - проверка версии гита
git init - инициализация папки
git status - состояние гита
git add - добавление содержимого
git commit -m '' - сохранение с описанием
git log - журнал изменений
git checkout **** - переключение между версиями
git checkout master - возвращение в сохранение, с которым работаем
git diff - разница между текущим файлом и сохраненным
